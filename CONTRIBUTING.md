# Contributing

Thanks for helping improve AI Agent Rules Generator.

## Good first contributions

- Add a project template
- Improve generated `AGENTS.md` wording
- Add an example for a real workflow
- Report confusing UI copy
- Suggest agent rules for a tool you use

## Pull request checklist

- The change is small and focused
- The app still works by opening `index.html`
- UI text fits on mobile and desktop
- Documentation is updated when behavior changes
- Generated output remains practical for maintainers

## Maintainer values

- Keep the project beginner-friendly
- Prefer plain HTML, CSS, and JavaScript
- Make the output useful enough to paste into a real repo
- Avoid hype; explain what the tool actually helps with
