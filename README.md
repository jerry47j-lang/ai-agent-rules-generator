# AI Agent Rules Generator

Generate repo instruction files for AI coding agents in under 30 seconds.

## Why this project is timely

AI coding agents are becoming a normal part of software work, but most repositories still do not give agents clear project context. This tool helps maintainers create the files agents and IDE assistants look for:

- `AGENTS.md`
- `.cursor/rules/project-rules.mdc`
- `.github/copilot-instructions.md`
- `CLAUDE.md`
- `launch-post.md`

## Run locally

Open `index.html` in a browser.

No build step, backend, account, or API key is required.

## Suggested GitHub repo description

Generate AGENTS.md, Cursor rules, Copilot instructions, and CLAUDE.md for AI coding agents.

## Launch angle

Most people ask AI agents better prompts. This project helps repos give agents better context.
